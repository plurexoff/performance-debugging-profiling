# НАЧАЛЬНЫЙ АНАЛИЗ ПОТОМЕбЛЕМ ПРОИЗВОДИТЕЛЬНОСТИ

## Основные проблемы

### 1. ПОИСК В ОНКОУНЮЧЕСОМ СПИСКЕ
- **Цена**: O(n²)
- **Причина**: Функция `in` на списке выполняет линейный поиск
- **Решение**: Оптимизация - использую структуру Set для O(1) основном поиска

### 2. НЕЭФФЕКТИВНАЯ FIBONACCI
- **Цена**: O(2^n) - экспоненциальная!
- **Причина**: Перевыслитываются одни и те же значения
- **Решение**: Итеративный подход O(n) или мемоизация

### 3. НЕЭФФЕКТИВНАЯ КОНКАТЕНАЦИЯ СТРОК
- **Цена**: O(n²) - строки неизменяемые
- **Причина**: Новая строка создается на каждом добавлении
- **Решение**: Коллектор с join() - O(n)

### 4. НЕОПОТОМАЛЬНАЯ СОРТИРОВКА
- **Цена**: O(n²) - бубыльная сортировка
- **режение**: Использование встроенного класса sorted() - O(n log n)
